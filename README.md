## Getting Started

```bash
go get https://github.com/vapply-labs/framework
```

## Getting Started [Contributor]

Make sure your go version is at least `go 1.19`.

```bash
go install ./...
```

Now, you can feel free to run tests or contribute!

```bash
# Basic
go test -v ./...
```
